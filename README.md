# Draw With Hasura

A pixel art app built with React and Hasura. Based off this [Hasura tutorial](https://welearncode.com/hasura-101/) by [Ali Spittel](https://www.alispit.tel/).

[Try out the app](https://draw-with-hasura.netlify.app/).
